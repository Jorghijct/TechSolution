# NetBeans - TechSolutions

## Descripción

Sistema web para la gestión de productos de una empresa de computación desarrollado como proyecto universitario.

## Características

- Formulario para alta de categorías
- Formulario para alta de productos
- Diseño responsive y moderno
- Validación de campos

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP
